---------------- Tutorial :: Upgrade From 1.2.x --------------
-------------------------------------------------------------------------
1. Copy overwritten all of content inside "upload" folder (include subs directories) into
"http://yoursite.com/Xoops_root_dir/" except file "conf_global.php".

From v1.3x, you don't need copy images folder, only *.php files are needed.

2. Copy files inside "upgrade_script" folder into "/xoops_root_dir/modules/ipboard/"
---|-- 1.2x  upgrade to 1.4x ==> Run "yoursite.com/modules/ipboard/upgrade6.php"
---|-- 1.3x  upgrade to 1.4x ==> Run "yoursite.com/modules/ipboard/upgrade7.php"

3. In XOOPS AdminCP,
--|--- Update System Module,
--|--- Passed to Step 5 if you are in fresh new install,

4. In IPB module's AdminCP,
--|--- Update XOOPS's BB Module => Common Settings => Submit ,

5. Finish.
